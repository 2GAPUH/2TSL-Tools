## ✨ Возможности

| Возможность | Описание |
|-------------|----------|
| 📋 **Шаблоны** | CRUD-менеджер текстовых шаблонов с группами, поиском и вставкой прямо в чат |
| 🔗 **Навигация** | Кликабельные ссылки на НЛС и номера заявок в Omnichat → мгновенный переход в TTM |
| 🔀 **Межсистемный мост** | TTM → Ассистент (автозаполнение формы), Onyma, SIPAL — одним кликом |
| 📊 **Учёт заявок** | Боковая панель в Omnichat: закрытые, выезды, статистика, CSV-экспорт |
| ⏱ **Напоминания** | Таймеры в TTM с уведомлениями Chrome, редактирование и автосборка |
| 🌐 **Grafana → SSH** | Кликабельные IP-адреса в дашбордах с автоматическим переходом на SSH-терминал |
| 🌙 **Тёмная тема** | Поддержка светлой и тёмной темы во всех интерфейсах |
| ⚙️ **Гибкие настройки** | 9 независимых переключателей для каждого модуля |

## 🗺 Карта навигации между системами

```
Omnichat (НЛС / Тикет)
    ├──→ TTM (автопоиск)
    │       ├──→ Форма ассистента (автозаполнение: номер, услуга, РФ, технология)
    │       ├──→ Onyma (автопоиск по ИЛС, начинается с 4)
    │       └──→ SIPAL (автопоиск по ИЛС, начинается с 2)
    └──→ Напоминания (таймер на TTM → уведомление в Chrome)

Grafana (IP-адреса)
    └──→ SSH-терминал (автозаполнение IP по региону)
```

## 🔧 Модули

### Popup — Всплывающее окно

Четыре вкладки в интерфейсе расширения:

- **Шаблоны** — создание, редактирование, удаление, группировка, поиск, копирование и вставка шаблонов в чат
- **Учёт заявок** — ежедневный трекинг закрытых заявок и выездов, статистика производительности, CSV-экспорт
- **Напоминалка** — просмотр, редактирование и удаление напоминаний
- **Настройки** — переключатели модулей, тёмная тема, управление данными

### Content Scripts — Интеграции

| Файл | Система | Функционал |
|------|---------|-----------|
| `content-omnichat.js` | `omnichat.rt.ru` | Вкладка «Дополнительно» с пользовательскими шаблонами; автолинковка НЛС и номеров заявок в чате |
| `content-accounting.js` | `omnichat.rt.ru` | Плавающая панель учёта заявок со статистикой и CSV-экспортом |
| `content-ttm.js` | `www.ttm.rt.ru` | Кнопки: Ассистент, Таймер, Onyma, SIPAL; автопоиск при переходе из Omnichat |
| `content-form.js` | `bzbti.rt.ru` | Автозаполнение формы «Использование ассистента» данными из TTM |
| `content-grafana.js` | `epd.rt.ru` | IP-адреса в таблицах Grafana → кликабельные ссылки на SSH (Волга, Юг, СЗ) |
| `content-ssh.js` | `10.x.x.x` | Автозаполнение IP на SSH-терминале |
| `content-onyma.js` | `onymaweb.south.rt.ru` | Автозаполнение ИЛС в поиске Onyma |
| `content-sipal.js` | `sipal.sz.rt.ru` | Автозаполнение НЛС в поиске SIPAL |
| `content-userinfo.js` | `lb.volga.rt.ru` | Автозаполнение MAC-адреса *(не зарегистрирован в манифесте)* |

### Background — Service Worker

- Регистрация пользователя и подсчёт активных установок
- Система напоминаний через `chrome.alarms` + `chrome.notifications`
- Маршрутизация сообщений между popup и content scripts
- Автовосстановление alarms при перезапуске браузера
- Периодическая очистка устаревших напоминаний

## 💾 Хранение данных

Все данные хранятся локально в `chrome.storage.local` — ничего не отправляется на внешние серверы (кроме анонимного ID для подсчёта пользователей):

| Данные | Назначение |
|--------|-----------|
| `templates`, `groups` | Пользовательские шаблоны и их группы |
| `settings` | Переключатели функций |
| `requestsByDate` | Данные учёта заявок по дням |
| `reminders` | Список напоминаний |
| `ttmFormData` | Данные для передачи TTM → Форма ассистента |
| `ttmSearchData` | Данные для передачи Omnichat → TTM |
| `onymaSearchData`, `sipalSearchData` | Данные для автопоиска в Onyma/SIPAL |
| `sshTransferData` | IP-адрес для передачи Grafana → SSH |

## 🚀 Установка

### Из Chrome Web Store (рекомендуется)

> Скоро будет доступно

### Вручную (Developer Mode)

1. Склонируйте репозиторий:
   ```bash
   git clone https://github.com/2GAPUH/2TSL-Tools.git
   ```
2. Откройте Chrome → `chrome://extensions/`
3. Включите **Режим разработчика** (переключатель в правом верхнем углу)
4. Нажмите **Загрузить распакованное расширение**
5. Выберите папку `2TSL-Tools`

### Обновление

Расширение поддерживает автообновление через собственный update-сервер. При ручной установке обновляйте через `git pull` и перезагрузку расширения на странице `chrome://extensions/`.

## 🏗 Технические детали

- **Чистый JavaScript** — без фреймворков, без сборщиков, без npm
- **Manifest V3** — современная архитектура Chrome Extension
- **CSS Variables** — поддержка светлой и тёмной темы
- **MutationObserver** — отслеживание динамического контента в SPA
- **Draft.js интеграция** — вставка текста в редактор Omnichat через ClipboardEvent
- **SPA-навигация** — патчинг `history.pushState` для отслеживания переходов в TTM
- **TTL-валидация** — автоматическая очистка межсистемных данных (30 сек)
